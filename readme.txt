The widget is designed to stimulate comments, trackbacks and pingbacks on your site, thereby raising the popularity of your site.

== Description ==

Features:
*  Order - sets whether new ones come first or old ones ome first.
*  Author, Date - sets whether item author, date will be inserted or not.
*  Style - sets the enclosing HTML tag from either ol, ul, or div.
*  Type - sets the comment type from either Comments, Pingbacks and Trackbacks, Pingbacks, Trackbacks or All.
*  Color - set custom color for smile, text inside of block



== Installation ==

Before using this theme, you have to install Wordpress. To install Wordpress, just follow instructions in the Wordpress Codex: Installing Wordpress

To install SmileBox plugin, there are 2 ways:

First way: In the Dashboard, navigate to Plugins. Click Add New Â» Upload and then upload the file nxIK-smile-box-widget.zip. After uploading completed, just activate the plugin.
Second way: Using FTP Upload. Filezilla is a free FTP Software. Extract the nxIK-smile-box-widget.zip file and upload the folder to directory /wp-content/plugins/ on your server.
